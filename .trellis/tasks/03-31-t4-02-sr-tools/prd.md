# T4-02: 支撑阻力绘制工具

## Goal
实现支撑阻力绘制工具和斐波那契工具，支持在图表上绘制、编辑、删除技术分析线。

## Requirements
- 水平线工具：绘制支撑/阻力水平线
- 趋势线工具：绘制上升/下降趋势线
- 平行通道工具：绘制平行通道
- 斐波那契回撤：绘制斐波那契回撤线（0%, 23.6%, 38.2%, 50%, 61.8%, 100%）
- 斐波那契扩展：绘制斐波那契扩展线
- 编辑功能：选中后可编辑位置
- 删除功能：选中后可删除
- 工具栏：工具选择和切换

## Acceptance Criteria
- [ ] SupportResistanceTools 组件正确渲染工具栏
- [ ] 水平线绘制功能正常工作
- [ ] 趋势线绘制功能正常工作
- [ ] 平行通道绘制功能正常工作
- [ ] 斐波那契回撤绘制功能正常工作
- [ ] 斐波那契扩展绘制功能正常工作
- [ ] 编辑和删除功能正常工作
- [ ] 工具状态正确持久化

## Technical Notes
- 基于 LightweightCharts 的 drawings API
- 参考 T4-01 完成的支撑阻力算法
- 参考 T3-01 完成的 ChartBase 组件
